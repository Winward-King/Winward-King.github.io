OpenWrt 管理界面安装中文语言包

  1. 打开 OpenWrt 的管理界面，选择「System」>「Software」。
  2. 选择「Update lists」更新软件列表
  3. 确认更新成功。
  4. 在筛选框中输入「luci-i18n-base-zh-cn」进行查找，接着选择「Install」安装语言包。
  5. 语言包安装成功，刷新页面后界面语言会替换成中文。

开启硬件Nat

启用硬件NAT

  1. 主要针对于千兆带宽用户
  2. 登陆路由器的网页。
    • Network -> Firewall -> General Settings -> Software flow offloading(勾上) -> Hardware flow offloading(勾上) -> Save&Apply(提交)
    • 路由/Nat分载 -> 软件流量分载(勾上才会出现硬件选项) -> 硬件流量分载(勾上)

软件包安装

  • 一定要安装以luci-app-xxxx开头，才能出现服务选项卡
