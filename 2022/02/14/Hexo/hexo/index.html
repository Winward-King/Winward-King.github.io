前言

上学时，博主一直拿印象笔记、word和onenode记笔记。上网冲浪时，发现博客查阅更加方便申请了博客园，但是不够定制化、稳定化，博客园也经历过丢博客、博客被撤销等事件，于是笔记重新转为硬盘存储。后来博主发现了一个很好的私人定制化方法，Hexo博客（Github 与 Hexo 的结合），可以推送到GIT、GITEE、VBS等平台进行多重备份，反向代理，于是就有了这次Hexo+GIT搭建的笔记。

软件与平台

必须的软件：

NODE.JS

GIT

可选的软件

Visual Studio Code（推荐）

TYPORA

安装说明

node.js 和 Typoro、Git，一路选 next 就行了。

VsCode 安装好之后默认是英文界面，可以在插件市场里搜索 chinese，安装之后重启软件，变成中文版。

检查是否安装成功

Windows下用 Win + R 打开运行，输入 cmd 并进入 cmd 窗口

Mac下打开终端，进行查询

或者在右键打开 Git bash，输入命令。能看到版本号就表示成功啦。

  • node.js 版本的检查
  • npm 版本的检查 (NPM 是随同 NodeJS 一起安装的包管理工具，能解决 NodeJS 代码部署上的很多问题)。
  • Git 版本的检查
